# Writing Coach App

A motivational writing coach web app for tracking daily writing progress, maintaining streaks, and practicing with focused writing exercises.

## Features

- **Daily Word Count Tracking**: Log your daily writing progress and watch it add up
- **Customizable Goals**: Set and adjust your daily word count target
- **Streak Counter**: Build momentum by maintaining consecutive writing days
- **Writing Exercises**: Random literary horror-focused writing prompts and exercises
- **Progress Visualization**: See your daily progress with an intuitive progress bar
- **Data Persistence**: All data saved locally in your browser
- **Motivational Messages**: Daily encouragement to keep you writing

## Live Demo

Visit the app at: `https://yourusername.github.io/writing-coach/`

## Technologies Used

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Babel Standalone (for JSX compilation)
- LocalStorage API (for data persistence)

## Local Development

To run locally:

1. Clone this repository
2. Open `index.html` in your browser
3. That's it! No build process needed.

## Customization

The app is fully customizable. Edit `index.html` to:

- Add or modify writing exercises
- Change motivational messages
- Adjust color schemes
- Modify default daily goals
- Add new features

## Browser Compatibility

Works in all modern browsers:
- Chrome
- Firefox
- Safari
- Edge

## Data Storage

All data is stored locally in your browser using LocalStorage:
- Word counts
- Streak information
- Daily goals
- Progress history

**Note**: Data is device and browser-specific. Using a different browser or device will start fresh.
